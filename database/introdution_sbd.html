Introdução a Sistemas de Banco de Dados


por Camila Cardoso, 20/05/2019

    Você sabe o que é um sistema de banco de dados? E o que é um banco de dados? E o que é um sistema gerenciador de banco de dados? Para explicar esses três pontos, vamos começar pela definição genérica sobre o que é um banco de dados sob a perspectiva de alguns autores::

    Banco de dados pode ser considerado um repositório ou recipiente para uma coleção de arquivos de dados computadorizados, que permitem diversas operações envolvendo tais arquivos(DATE C J, 2003). Em outra linguagem, pode ser definido como um conjunto de dados relacionados. Podemos compreender como dados os objetos conhecidos que podem ser armazenados e que possuem um significado implícitos. Um banco de dados possui as seguintes propriedades:


       - É uma coleção logicamente coerente de dados com algum significado inerente;


       - Representa algum aspecto do mundo real, às vezes chamado de mini-mundo ou de universos de discurso (UoD - Universe of Discourse);


       - É projetado, construído e populado com dados para uma finalidade específica (Elmasri e Navanthe, 2011).


    Já um Sistema Gerenciador de Banco de Dados (SGBD), ainda segundo Elmasri e Navanthe, é um conjunto de programas que permite aos usuários criar e manter um banco de dados. Definir um banco de dados envolve especificação de tipos, estruturas e restrições dos dados que serão armazenados. A definição ou informação descritiva do banco de dados também é armazenada pelo SGBD na forma de um catálogo ou dicionário, chamado de metadados.


    Por fim, o sistema de banco de dados é a união do banco de dados com o software SGBD.


REFERÊNCIAS BIBLIOGRÁFICAS


Elmasri e Navanthe. Sistemas de Banco de Dados: 6 ed. São Paulo: Editora Pearson Addison, 2011.

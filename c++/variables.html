Variáveis em C++


por Camila Cardoso, 09/05/2019

Já foi introduzido uma pequena explicação sobre a história da Linguagem C++ no tópico anterior, além disso, também foi feito feito um código ensinando o “Hello World!”, agora vamos entrar em outro assunto que são as variáveis em C++.


Você sabe o que é uma linguagem de programação tipada? Se não sabe, vou explicar aqui. As linguagens de programação tipadas são aquelas que possuem um tipo bem definido e que precisam ser especificados. A Linguagem C++ se enquadra nas linguagens tipadas, inclusive é definida como uma linguagem fortemente tipada, é importante salientar que existem classificações dentro das linguagens tipadas, mas não vamos entrar no mérito deste assunto agora. A linguagem fortemente tipada segue o mesmo conceito já dito, ou seja, isso significa que cada variável possui seu tipo e necessita especificação.


Agora que já entendemos um conceito básico que envolve as variáveis, falaremos um pouco sobre os tipos que existem em C++.


Os principais tipos de variáveis


charArmazena um caractere ou um inteiro pequeno.
short intArmazena valor de 0 a 65.535.
intArmazena valor de 0 a 4294967295.
long intArmazena valor de 0 a 4294967295.
boolTrue ou False.
floatArmazena valores de 1.2e-308 a 3.4e-38.
doubleArmazena valores: de 2.2e-308 a 3.4e-38.

Então, podemos dizer que a declaração em C++ se faz da seguinte maneira:

{tipo da variável} nome = {valor correspondente ao tipo};

Exemplos de usabilidade das variáveis em C++:


int numero = 10;


bool condicao = false;


char caractere = 1;


float total = 4.58;


Não é um assunto difícil de entender, pode até parecer meio bobo, mas é necessário, ainda mais quando estamos falando de uma linguagem fortemente tipada.

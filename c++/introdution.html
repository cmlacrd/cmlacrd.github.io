Introdução a linguagem C++


por Camila Cardoso, 06/05/2019

HISTÓRICO


Por volta de 1983, a linguagem C++ foi criada, por Bjarne Stroustrup, baseada na linguagem C e em Simula. É uma das linguagens mais populares orientada a objetos. Segundo Saade, C++ é uma linguagem de propósitos gerais com uma tendência para a programação de sistemas, que suporta programação de baixo nível nos moldes tradicionais, abstração de dados e orientação a objetos.


ESTRUTURA DE UM CÓDIGO C++


Agora vamos escrever um código simples, vamos fazer o famoso "Hello World" para aparecer na tela. Vamos dar o nome do arquivo de "hello_world.cpp" e ele vai estar assim:


#include < iostream >


using namespace std;


int main() {


cout << "Hello World" << endl;


return 0;


}


Para entender melhor o código vamos explicar linha por linha com a seguinte imagem:



O código sem o uso do using namespace std ficaria assim:

#include < iostream >


int main() {


std::cout << "Hello World" << std::endl;


return 0;


}


Agora basta executar o arquivo em sua máquina e, isso, vai depender do sistema operacional que está utilizando. Se tiver utilizando Linux, basta abrir o terminal na pasta que o arquivo C++ se encotra e escrever o seguinte comando:


g++ hello_world.cpp -o hello


Se não tiver nenhum erro retornando, basta escrever:


./hello


Observe que depois do -o eu denominei o nome que eu queria por para executar o programa e escolhi "hello". Por isso, utilizamos o './hello'. Você pode escolher qualquer nome para a execução do seu programa.


C++ é uma linguagem bastante simples. Ao longo desta seção vamos aprofundar cada vez mais nessa linguagem e usufruir dela da melhor maneira.
